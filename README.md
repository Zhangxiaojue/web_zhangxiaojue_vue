#### 下载依赖

```
npm install
```

#### 编译并热加载以供开发

```
npm run dev
```

#### 为生产而编译和缩小

```
npm run build
```

#### 清理并修复文件

```
npm run lint
```

See [Configuration Reference](https://cli.vuejs.org/config/).

#### 代码提交并上传

```
1. git pull 同步远程仓库代码到本地
2. git status 查看当前状态
3. git add . 提交代码到本地 git 缓存区
4. git commit -m "提交代码" 推送修改到本地 git 库中
5. git push origin <远程分支名> 提交本地代码到远程仓库
```
